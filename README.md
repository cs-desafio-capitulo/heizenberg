# About
Heizenberg is the API for managing the products/items.

# Endpoints
* POST `/products/create`
* GET `/products/{id}`
* PUT `/products/{id}`
* DELETE `/products/{id}`

# Stack
## Dependencies
* node 8.9.3 (lts)
* npm 5.5.1
* express
* mongoose

## Dev dependencies
* babel
* chai
* eslint
* mocha
